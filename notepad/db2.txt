第一步：下载镜像 
docker pull ibmcom/db2express-c:latest

第二步：启动镜像 
docker run -d --name dbii -p 50000:50000 -e DB2INST1_PASSWORD=root -e LICENSE=accept ibmcom/db2express-c:latest db2start

docker exec -it dbii /bin/bash


su db2inst1

db2 create db SP using codeset UTF-8 territory CN


whoami

db2 connect to AUDIT


第五步：DBeaver工具可以连接DB2数据库 
用户名: db2inst1 
数据库名： AUDIT
密码：[数据库密码]
端口：5000
第六步：给数据库创建数据库管理员（只要执行6个命令）
命令解释：

a. 进入镜像 

docker exec -it db2 /bin/bash
b.在镜像里面 添加用户 
useradd -g users -d /home/mtdb2dev -s /bin/bash -m mtdb2dev
c.修改用户密码
 

passwd mtdb2dev
d.切换到 db2inst1用户
su db2inst1
e.链接目标数据库 
db2 connect to mtdb2DB
f.给数据库授权用户 
db2 grant dbadm on database to user mtdb2dev
